Requiere iOS

Apple Health

HealthKit no funciona desde una web normal. Hace falta app iOS para leer pasos, calorias, sueño y entrenamientos.

MVP que hay que construir primero

1Auth real: registro, login, reset y eliminar cuenta.
2Onboarding persistente con perfil, objetivos, lesiones, disponibilidad y preferencias.
3Base de datos por usuario para rutinas, entrenos, comidas, peso, medidas y habitos.
4Rutina semanal generada por objetivo, nivel, días, tiempo y material.
5Modo entreno con historial real y progresión automática.

Arquitectura recomendada

React + TypeScript Firebase Auth Firestore por usuario Storage privado para fotos React Native o SwiftUI para HealthKit IA de comida con revision manual obligatoria
